# BitBattle
Simple Unity 2D platformer game

  Bitbattle is a 2D platformer game made in Unity. It uses a 8-bit style reminiscent of early video games. A video playthrough of Bitbattle can be found at [bitbattle_recording](/bitbattle_recording.mov)

The bitbattle_part_* files can be recombined using `cat bitbattle_part_* > bitbattle.zip` to reconstruct the full game's .zip file.
